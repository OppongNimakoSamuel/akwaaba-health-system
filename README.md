🏥 Akwaaba Health System

A hospital management system for managing patients, staff, and appointments efficiently.

🚀 Features
Staff & Patient Management
Appointment Scheduling
Dashboard Monitoring
Secure Login System with Multi-Factor Authentication (MFA)
🔐 Login Instructions (IMPORTANT FOR TESTING)

This system uses a two-step authentication process.

✅ Step 1: Initial Login

Enter the following credentials:

Staff ID: ADMIN
Password: 1234
Multi-Factor Token: (leave empty)

Click Access Dashboard

🔁 Step 2: Multi-Factor Authentication

After successful login, the system will redirect you to a second login page.

Now enter:

Staff ID: ADMIN
Password: 1234
Multi-Factor Token: (enter the token displayed on screen)

Click Access Dashboard again.

⚠️ Important Notes
The MFA token is generated after the first login attempt
Login will fail if the token is not entered correctly
You must enter all three fields correctly in the second step
